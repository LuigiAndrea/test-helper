# Go Test Helper

## Install and Usage

### Install the package with
```bash
go get github.com/LuigiAndrea/test-helper
```
### Import it with
```go
import (
"github.com/LuigiAndrea/test-helper/assertions"
"github.com/LuigiAndrea/test-helper/messages"
)
```
